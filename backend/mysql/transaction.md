## 2. 学习目标

本主题不是背 ACID 定义，而是建立事务主线：

1. 为什么多个 SQL 操作需要被绑定成一个整体
2. 原子性、一致性、隔离性、持久性分别解决什么问题
3. 并发事务为什么会产生脏读、不可重复读、幻读
4. 为什么需要隔离级别
5. MVCC 为什么出现
6. undo log、redo log 分别解决什么问题
7. 事务和业务幂等、状态机、补偿有什么边界
